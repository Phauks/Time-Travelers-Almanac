# The Time Traveller's Almanac

A cheeky, *Hitchhiker's-Guide*-style field guide to time travel and time loops
in film, TV, books, games, and stage — for anyone who's become unstuck in time
and needs to work out **what kind of temporal predicament they're in** and
**what they're allowed to do about it.**

Planned as a Svelte + TypeScript site, hosted on GitHub Pages.
